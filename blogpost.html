Cabecera Mi blog usando protocolo SSH

Comandos básicos en git usando protocolo SSH

  • Inicializa git en el directorio actual (si ya está iniciado lo reinicia):

    git init

  • Muestra qué archivos hay y su estado (tracked, untracked):

    git status

Al escribir este párrafo regresé a la rama master donde el blog no tiene diseño en el archivo css, pero en la rama "cabecera" están todas las características de diseño.
